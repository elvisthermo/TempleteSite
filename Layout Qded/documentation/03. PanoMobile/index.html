FAQ

How do I install the item?

Copy and paste all the items in the code folder to your server to your desired location. 

What are those i class"fa " things?

That is Font Awesome, the amazing icons we used for this item you can learn more about them and find a cheatsheet on how to use them here http://fortawesome.github.io/Font-Awesome/icons/

How do I change the contact form information?

There is only one PHP file in the item, please open that item and replace the email address with yours. 

How do I make the login page / sign in page / calendar page to work? 

Please remember that this is a site template and is not connected to any CMS or any databases, so these features are mearly template designs, they are HTML and CSS and need to be connected to a back-end to work. 

The charts aren't loading, what am I doing wrong?

For the charts to work you must add an aditional body class to your page, so for example 

<body class="POSITION-sidebar has-charts"> the has charts class will enable the javascript that loads the charts. This is essential for performance reasons! 

Header Style

Please don't forget to follow the structure presented in the first Documentation ( STRUCTURE - VERY IMPORTANT ) 

 

Page Header

<div class="homepage-header">
            <a href="#" class="show-navigation"><i class="fa fa-navicon"></i></a>
            <a href="contact.html" class="show-contact"><i class="fa fa-envelope"></i></a>
            <a class="homepage-logo"></a>
        </div>
<div class="header-line container-fullscreen"></div>

The header is simple to use and very easy to modify to suit your needs. 

Line 1 is the hosing class, homepage-header is the name of the class although we used it throught the pages

Line 2 is the navigation button, the class show-navigation activates the menu

Line 3 is the contact icon, the class has no value other than positioning it

Line 4 is the header logo. To edit it, open style.css and go to line 281. Replace the image there with your owna nd make sure to change the background-size, height and width to half the size of your logo to maintain retina quality. 

 

 

 

 

 

 

Navigation Elements

Please don't forget to follow the structure presented in the first Documentation ( STRUCTURE - VERY IMPORTANT ) 
 
Menu Header

The menu header consists of the logo, which can be replaced in the same way as the header logo on line 73, and the subtitle of the logo, we don't know, maybe you have an awesome catchphrase you want to use! :) 

 

Simple Menu

A simple menu can be generated using the following sturcutre. Simple menus are direct links with icons.  Having the secondary class active-navigation-item will change the circle at the end to green and increase the contrast of the menu. 

    <a href="index.html" class="active-navigation-item navigation-item">
        <i class="fa fa-home"></i>
            <em>Homepage</em>
        <i class="fa fa-circle"></i>
    </a>

 

Dropdown Menu

Dropdown menus have the same structure as the normal menu, only the include a housing div the submenu class, and the main anchor that deploys the submenu must have the class open-submenu. If you want the submenu to be activated by default, you must add the class rotate-icon to the main div, as shown on line 5, and also on line 7 have the secondary class show-submenu. To activate any submenu, the same class applies active-navigation-item.

 

    <div class="submenu">
        <a href="#" class="open-submenu navigation-item">
            <i class="fa fa-cog"></i>
                <em>Features</em>
            <i class="fa fa-plus rotate-icon"></i>
        </a>
        <div class="submenu-items show-submenu">
            <a href="features-typography.html" class="navigation-item active-navigation-item">
                <em>Typography</em>
                <i class="fa fa-circle"></i>
            </a>              
            <a href="features-jquery.html" class="navigation-item">
                <em>jQuery</em>
                <i class="fa fa-circle"></i>
            </a>             
        </div>
    </div>

 

Changing the default height of the menu.

The menu by default is scrollable, that means if you open a submenu you can easily scroll inside it, but if you would like to increase the height it currently has ( warning, it will be cropped on smaller devices such as the 5s, we calibrated the menu specifically to work on everything ) you can do so by opening style.css and go to line 41 and replace the height parameter to a bigger value. Be sure to change the margin-top value to NEGATIVE half the size of the height you will use.
